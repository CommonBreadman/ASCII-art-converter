# Jyrk's simple ASCII art converter
This is a very simple ASCII art converter, made using C++ and STB_Image
## Instructions
Compile the project, put the binary in the same folder as an image (png or jpg) file of your choice, start up the binary, type in the file name and desired width, and enjoy your converted image.
### IMPORTANT: THE ASCII ART ALSO APPEARS IN A NEW "results.txt" FILE, WHICH GETS *WIPED ON EVERY USE*
